Tensorflow.js Image Classifier

  • This is a basic template layout of a deployable implementation of a pretrained neural network as a web application.
  • The pretrained model used here is the keras.applications.mobilenet.
  • This model is lightweight and can be trained and used to infer labels for new images on the browser itself without any external support.

    • Allow browser access to your webcam.

    • Click atleast 30 pictures for each class
    • After the training you can show one of 4 classes to the webcam and you can see prediction and probability at the Output section.
